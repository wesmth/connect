Bem vindo ao API Connect

Sobre o Projeto – API Connect:

O API Connect é um projeto pessoal desenvolvido com o objetivo de colocar em prática os conhecimentos que venho adquirindo ao longo dos meus estudos em programação, tanto por meio de cursos formais quanto por aprendizado autodidata. Essa iniciativa surgiu nos primeiros meses de 2025, como parte do meu processo de evolução como desenvolvedor e também como forma de documentar e consolidar o que venho aprendendo na área. Durante o desenvolvimento, utilizei principalmente o framework Django e a linguagem Python, que foram as bases para estruturar e dar vida ao projeto. Além disso, ao longo do caminho, tive contato com outras tecnologias importantes, como HTML, CSS e o Bootstrap, que me ajudaram a tornar a interface mais agradável e responsiva. Esse processo de construção me permitiu não apenas aplicar conhecimentos já adquiridos, mas também explorar novos conceitos e ferramentas. O foco principal do projeto é exercitar a integração com APIs RESTful públicas, utilizando dados reais em um ambiente de teste, de forma prática e funcional. Além disso, integrei a IA Gemini, responsável por gerar respostas personalizadas e interativas, o que agregou ainda mais dinamismo à aplicação. O API Connect é, acima de tudo, um projeto de aprendizado contínuo. Ele foi pensado para ser parte do meu portfólio, demonstrando minha capacidade de construir soluções completas com back-end, front-end e integração com serviços externos. Mais do que um simples exercício, ele representa uma etapa importante no meu desenvolvimento como profissional da área de tecnologia.

Descrição das Páginas do API Connect

Página Inicial

Aqui você encontra uma visão geral do projeto: o que é, quais tecnologias foram usadas e qual o objetivo. É uma introdução simples e objetiva para quem quer entender rapidamente do que se trata o API Connect.

Desculpas Esfarrapadas

Nessa página, o usuário escolhe um tema,como trabalho, escola ou família, e a IA Gemini gera uma desculpa totalmente personalizada e criativa na hora. Não são respostas prontas, tudo é criado dinamicamente com base no input do usuário, mostrando como a inteligência artificial pode ser integrada para gerar conteúdo original.

Gato Falante

Aqui você digita uma frase e uma API pública retorna uma imagem de um gato com essa frase estampada. É uma forma prática e divertida de demonstrar o consumo de APIs que retornam conteúdo visual dinâmico, com uma interface simples e agradável.

Sim, Não ou Talvez

Essa página permite que o usuário faça qualquer pergunta e receba uma resposta aleatória: “sim”, “não” ou “talvez”. A resposta vem da API externa, é traduzida no backend e acompanhada por um GIF que combina com a resposta, proporcionando uma experiência interativa e leve.

Por que esse projeto importa?

Porque ele demonstra a aplicação prática de várias tecnologias: Django, APIs RESTful, IA generativa e front-end, de forma integrada e funcional. Cada página representa um desafio diferente, mostrando habilidades técnicas e criatividade, tudo isso em um projeto simples e acessível.
