Sobre o Projeto – API Connect:

O API Connect é um projeto pessoal criado com o objetivo de aplicar, na prática, os conhecimentos adquiridos ao longo dos meus estudos em programação — tanto por cursos formais quanto por aprendizado autodidata. Essa iniciativa surgiu nos primeiros meses de 2025, como parte do meu processo de evolução como desenvolvedor e também para documentar e consolidar o que venho aprendendo.

Durante o desenvolvimento, utilizei principalmente o framework Django e a linguagem Python, que serviram como base para estruturar e dar vida ao projeto. Além disso, explorei tecnologias como HTML, CSS e Bootstrap, que contribuíram para uma interface agradável e responsiva.

Esse processo permitiu não apenas aplicar conhecimentos adquiridos, mas também explorar novos conceitos e ferramentas. O foco principal do projeto é exercitar a integração com APIs RESTful públicas, utilizando dados reais em um ambiente de teste funcional. Também integrei a IA Gemini, responsável por gerar respostas personalizadas e interativas, agregando dinamismo à aplicação.

O API Connect é, acima de tudo, um projeto de aprendizado contínuo. Foi pensado para ser parte do meu portfólio, demonstrando minha capacidade de construir soluções completas com back-end, front-end e integração com serviços externos. Mais que um exercício, representa uma etapa importante na minha jornada como profissional de tecnologia.

Descrição das Páginas do API Connect

Página Inicial

Apresenta uma visão geral do projeto: o que é, quais tecnologias foram utilizadas e qual o objetivo. É uma introdução simples e direta para quem deseja entender rapidamente do que se trata o API Connect.

Desculpas Esfarrapadas

Nesta página, o usuário escolhe um tema — como trabalho, escola ou família — e a IA Gemini gera, em tempo real, uma desculpa totalmente personalizada e criativa. As respostas são criadas dinamicamente com base no input do usuário, mostrando o potencial da inteligência artificial para gerar conteúdo original.

Gato Falante

O usuário digita uma frase e uma API pública retorna uma imagem de um gato com essa frase estampada. Uma forma divertida de demonstrar o consumo de APIs que retornam conteúdo visual dinâmico.

Sim, Não ou Talvez

Permite que o usuário faça qualquer pergunta e receba uma resposta aleatória: “sim”, “não” ou “talvez”. A resposta, obtida via API externa, é traduzida no backend e acompanhada por um GIF temático, proporcionando uma experiência leve e interativa.

Por que este projeto importa?

Porque demonstra, de forma prática e acessível, a aplicação integrada de diversas tecnologias — Django, APIs RESTful, IA generativa e front-end. Cada página representa um desafio técnico diferente, refletindo criatividade, lógica e domínio técnico.
